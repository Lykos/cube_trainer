Overusing Resources

The site is intended to be used by human users. Creating scripts or other automatic methods to access the site at a rate that human users couldn't is forbidden. Additionally, doing actions that trigger emails (account creating, password reset etc.) should be done at a reasonable level. Triggering more than 10 emails per day per human user is not reasonable. This includes triggering emails via multiple user accounts.
