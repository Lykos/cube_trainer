Cookie Policy

What is a Cookie

A cookie is a small piece of text that the browser stores. Every time a user accesses a website, all cookies belonging to this site are sent to the server. See Wikipedia for more information.

What is Local Storage

Local storage is a component of the browser that stores information for a website. The information stored is similar to cookies, but it's a more modern mechanism that has enhanced functionality. See Wikipedia for more information.

Cookies used on CubeTrainer

CubeTrainer uses a cookie called {{consentCookieKey}} to remember that the user has consented to using cookies. The user has to consent to cookies when registering the account and CubeTrainer only sets these after account registration. Furthermore, CubeTrainer uses a cookie called _cube_trainer_session. It was originally used to track the login status. Nowadays it serves no purpose, and we are working on removing it.

CubeTrainer uses Google Analytics to understand how users use the website. Google analytics sets two cookies starting with _ga.

Local Storage used on CubeTrainer

CubeTrainer uses local storage to remember which users are logged in. See angular-token for more information about the exact values that are stored.
