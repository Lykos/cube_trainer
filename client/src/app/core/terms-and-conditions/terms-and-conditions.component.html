Terms and Conditions

Sharing Accounts

Sharing Accounts between multiple users is forbidden. The rationale is that sharing alg sheets between users may be tricky for copyright reasons. Note that having multiple accounts is allowed as long as the number of accounts is reasonable. See overusing resources below for more details.

Hacking

Any sort of hacking to deteriorate the functionality of the site or access private information of other users is strictly forbidden.

Overusing Resources

The site is intended to be used by human users. Creating scripts or other automatic methods to access the site at a rate that human users couldn't is forbidden. Additionally, doing actions that trigger emails (account creating, password reset etc.) should be done at a reasonable level. Triggering more than 10 emails per day per human user is not reasonable. This includes triggering emails via multiple user accounts.

Security & Privacy Issues

Users agree to report any Security and Privacy Issue they discover as quickly as possible. If you discover a security bug, please report it via {{newSecurityBugLink}} or email {{securityBugEmail}}. We will respond within 14 days and either fix the bug or take the website offline until we have time to fix it. Note that this only counts for security bugs, normal bugs should be reported via Github.
